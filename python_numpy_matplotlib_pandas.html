3. Matplotlib¶

Ví dụ vẽ biểu đồ hàm số $y = x^2$

In [ ]:
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-20, 20, 1000)
y = x * x

plt.plot(x, y)
plt.title("Đồ thị y = x^2")
plt.xlabel("x")
plt.ylabel("y")
plt.show()

Ví dụ vẽ đồ thị SIN và COS

In [ ]:
x = np.arange(0, 3 * np.pi, 0.1)
y_sin = np.sin(x)
y_cos = np.cos(x)

plt.plot(x, y_sin)
plt.plot(x, y_cos)
plt.xlabel('Trục X')
plt.ylabel('Trục Y')
plt.title('Hàm SIN và COS trong khoảng [0, 3π]')
plt.legend(['SIN(x)', 'COS(x)'])
plt.show()

4. Pandas¶

Ví dụ đọc dữ liệu từ CSV và hiển thị

In [ ]:
import pandas as pd

# Tạo DataFrame từ dictionary
data = {
    'Tên': ['An', 'Bình', 'Chi'],
    'Tuổi': [20, 21, 19],
    'Điểm': [8.5, 7.0, 9.0]
}
df = pd.DataFrame(data)

print("----- DataFrame -----")
print(df)

# Tính trung bình điểm
print("\nĐiểm trung bình:", df['Điểm'].mean())
